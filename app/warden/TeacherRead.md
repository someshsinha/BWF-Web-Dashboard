# Teacher Module

This folder contains all the functionality related to the **Teacher role** in the application.

It handles the pages, components, and features that are specifically used by teachers within the dashboard.

## Current Features

For now, the teacher module includes the following features:

1. **Teacher Dashboard** – Provides an overview interface for teachers.
2. **Basic Navigation Structure** – Initial setup for teacher-related pages.

## Future Scope

More teacher-related features will be added here as development progresses, such as:

* Class management
* Assignment management
* Student interaction tools

All files and folders inside this directory should be related only to **teacher-specific functionality**.
